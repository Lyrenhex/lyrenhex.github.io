Technical information

This website is programmed by hand, using only Compound for a 3rd Party Tool. This process is done to simplify and speed up website development, allowing for the site to be broken up in the source into many smaller files, before being stitched up into one large, minified production file on release.

The website has also been created following Google's “Mobile First” website development guidelines, and is then gracefully upgraded via the use of free JavaScript to create the desktop edition of the site. This upgrade occurs when the JavaScript event handler, which is attached to the window resize event, detects that the window's width is greater than, or equal to, 1,000 pixels. Therefore, should JavaScript be disabled, a mobile edition of the site will be served regardless of screen size. This edition of the site removes modals, displaying all information sequentially in a rather long webpage.
